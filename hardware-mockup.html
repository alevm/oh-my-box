Technical Specifications

Audio Engine

  • Sample Rate: 44.1kHz
  • Bit Depth: 24-bit
  • Latency: <10ms
  • 8 Sample Slots (30s each)
  • 8-Track Step Sequencer
  • 24 Effect Types

Inputs

  • Stereo Line In (3.5mm TRS)
  • Microphone In (3.5mm TRS)
  • MIDI In (TRS-A)
  • USB-C (Data + Power)
  • GPS Antenna (internal)

Outputs

  • Stereo Main Out (3.5mm TRS)
  • Headphone Out (3.5mm TRS)
  • MIDI Out (TRS-A)
  • USB Audio (Class Compliant)

Controls

  • 8x Touch Pads (MPR121)
  • 8x Rotary Encoders
  • 5x Function Buttons
  • 8x RGB LED Indicators

Connectivity

  • WiFi 802.11 b/g/n
  • Bluetooth 4.2
  • GPS (NEO-6M)
  • AI API (Claude)

Physical

  • Size: 250 x 180 x 50mm
  • Weight: ~650g
  • Display: 7" IPS Touch
  • Battery: 5-7 hours
